# Inter font

These woff2 files are the Latin subset of the Inter typeface (weights 400,
500, 600, and 700, regular style), vendored locally so the app has no
runtime dependency on the Google Fonts CDN.

Inter is licensed under the SIL Open Font License, Version 1.1. See
https://github.com/rsms/inter/blob/main/LICENSE.txt for the full license
text.
